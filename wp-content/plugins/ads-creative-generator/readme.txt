== Description ==

Ads Creative Generator is a powerful WordPress plugin that analyzes business websites and generates professional ad creative ideas using artificial intelligence. Perfect for marketers, agencies, and business owners who want to create compelling ad campaigns.

**Key Features:**

* **Website Analysis**: Analyzes business websites to extract key insights
* **AI-Powered**: Uses OpenAI to generate professional ad creatives
* **Multiple Categories**: Creates creatives based on UVP, ICP, Pain Points, Triggers, Goals, and Benefits
* **Copy to Clipboard**: Easy copying of generated creatives
* **Daily Limits**: Configurable usage limits per user
* **Responsive Design**: Works on all devices
* **Admin Dashboard**: Complete management interface

**Analysis Results Include:**

* **UVP (Unique Value Proposition)**: Main advantage that distinguishes from competitors
* **ICP (Ideal Customer Profile)**: Portrait of ideal buyer
* **Pain Points**: Customer problems and frustrations
* **Triggers**: Situations that prompt action
* **Goals**: What customers want to achieve
* **Benefits**: Value customers get from the product

**Generated Creatives Include:**

* **Image Ideas**: Visual concepts for ad images
* **CTA for Image**: Call-to-action text for images
* **Headlines**: Short, compelling headlines
* **Ad Text**: 2-3 sentence ad copy with problem-solution-CTA structure

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ads-creative-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'Ads Creative Generator' in the admin menu
4. Configure your OpenAI API key in the Settings page
5. Use the shortcode `[ads_creative_generator]` on any page or post

== Frequently Asked Questions ==

= How do I get an OpenAI API key? =

You can get an API key from the OpenAI Platform at https://platform.openai.com/api-keys

= How do I display the plugin on my website? =

Use the shortcode `[ads_creative_generator]` on any page or post. You can also customize it with parameters:
`[ads_creative_generator title="Custom Title" button_text="Custom Button"]`

= Is there a daily limit? =

Yes, by default users can perform 3 analyses per day. This can be configured in the plugin settings.

= Do I need to be logged in to use the plugin? =

Yes, users need to be logged in to use the analysis feature.

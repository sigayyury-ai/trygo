# Ads Creative Generator - WordPress Plugin

## Описание плагина
Плагин для генерации креативов рекламных объявлений с возможностью настройки различных параметров и автоматического создания визуального контента.

## Основные функции (предварительно)
- Генерация текстовых объявлений
- Создание визуальных элементов
- Настройка таргетинга
- Экспорт готовых креативов
- Шаблоны для разных платформ

## Технические требования
- WordPress совместимость
- Адаптивный интерфейс
- Интеграция с популярными рекламными платформами
- Возможность кастомизации

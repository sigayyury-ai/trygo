# Архитектор проекта - Ads Creative Generator

## Общая информация
- **Название плагина**: Ads Creative Generator
- **Тип**: WordPress Plugin
- **Статус**: Прототип создан, ожидает согласования
- **Дата создания**: 2024

## Описание функционала
Плагин для анализа бизнеса клиента и генерации идей для рекламных креативов на основе ChatGPT.

## Архитектура интерфейса

### Экраны приложения
1. **Экран 1 - Ввод данных**
   - Поле для ввода адреса сайта
   - Кнопка "Анализировать сайт"
   - Индикатор загрузки
   - Функция запоминания URL при обновлении страницы

2. **Экран 2 - Результаты анализа**
   - Отображение анализируемого URL
   - Кнопка "Назад к анализу"
   - 6 секций с результатами анализа:
     - UVP (Уникальное ценностное предложение)
     - ICP (Идеальный клиент)
     - Боли ICP
     - Триггеры
     - Цели
     - Бенефиты

### Технические особенности интерфейса
- **Адаптивный дизайн**: Поддержка мобильных устройств
- **Локальное хранение**: URL сохраняется в localStorage
- **Анимации**: Спиннер загрузки, плавные переходы
- **UX**: Интуитивно понятный интерфейс с четкой навигацией

## Интеграции

### ChatGPT API
- **Назначение**: Анализ сайта и генерация бизнес-инсайтов
- **Входные данные**: URL сайта
- **Выходные данные**: Структурированный анализ в 6 категориях
- **Формат ответа**: Короткий текст, по делу

### WordPress Integration
- **Админ-панель**: Интеграция в WordPress admin
- **База данных**: Сохранение истории анализов
- **Пользователи**: Привязка к WordPress пользователям

## Технический стек

### Frontend
- **HTML5**: Семантическая разметка
- **CSS3**: Адаптивные стили, анимации
- **JavaScript**: Интерактивность, AJAX запросы
- **WordPress Admin API**: Интеграция с админкой

### Backend
- **PHP**: Основной язык плагина
- **WordPress Hooks**: Actions и Filters
- **WordPress Database API**: Работа с БД
- **cURL**: Запросы к ChatGPT API

### Дополнительные технологии
- **Node.js**: Возможные серверные задачи
- **Python**: Обработка данных (при необходимости)

## База данных

### Таблицы
- `wp_ads_creative_analyses` - история анализов
  - id, user_id, website_url, analysis_data, created_at

### Опции WordPress
- `ads_creative_chatgpt_api_key` - API ключ ChatGPT
- `ads_creative_settings` - настройки плагина

## Безопасность
- **Валидация URL**: Проверка корректности введенного адреса
- **Sanitization**: Очистка всех пользовательских данных
- **Nonce**: Защита от CSRF атак
- **Capabilities**: Проверка прав доступа пользователей
- **API Key**: Безопасное хранение ключей

## Производительность
- **Кэширование**: Кэш результатов анализа
- **Асинхронность**: AJAX запросы без перезагрузки страницы
- **Оптимизация**: Минификация CSS/JS файлов
- **Лимиты**: Ограничение частоты запросов к ChatGPT API

## Локализация
- **Поддержка языков**: Русский (по умолчанию), английский
- **WordPress i18n**: Использование функций перевода
- **Файлы переводов**: .po/.mo файлы
